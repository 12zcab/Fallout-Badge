# Soupa Cool Badge for FallOut:D

Erm yah why not build a Badge with Z80 instead of ESP32 :)

Slack Username:12zcab

# What it can do?

1.Display ur Detail on 1602 LCD

2.use joystick and 6 Buttons to control the device

3.Plug it onto ur computer and have more interaction through console!

AND THE MOST IMPORTANT POINT IS: ITS HACKABLE!!!!!!!!!

U can easily reprogram it by overriding data on the EEPROM

# Main Components
Pls refer to BOM.csv for more details
1.Z80 10MHz
2.Z80 PIO 10MHz IC
3.MS6850 Serial IC
4.WC1602A LCD Display
5.Push Buttons
6.5 Direction Mini Joystick
7.on Board CH340
